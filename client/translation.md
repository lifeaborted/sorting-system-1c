# Язык
Для генерации языка необходимо сперва войти в venv, созданный через qt creator.

Дальше, для создания/обновления исходного файла перевода необходимо ввести комманду ниже, где [ЯЗЫК] - язык на который будет переведен исходный.
```bash
mkdir translation
pyside6-lupdate . -ts translation/[ЯЗЫК].ts
```

После открываете QtLinguist, выбираете файл, созданный выше (```translation/[ЯЗЫК].ts```).

Если вы открываете файл перевода впервые, в открывшемся окне, в Source language выбрать такие параметры:
```json
Laungage: "Russian(русский)",
Country/Region: "Russia(Россия)"
```
В Target language выбрать соответствующий язык.

Можете начать переводить :>.

После перевода необходимо добавить (если созданы новые языки) файлы в ```.pyproject```.

И скомпилировать файлы перевода
```bash
pyside6-project build .
```

Переведённые файлы с расширением ```.qm``` добавить в ```resources.qrc```. При этом убедитесь, что файл располагается в translation.
